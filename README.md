# 디어마이디어

## 상명대학교 알리미 서비스
학교에서 학생들에게 제공되는 정보들을 카카오톡 플러스친구를 통해서 편하게 보여주기 위해서 만들었다.  
django를 이용해 rest api 서버를 구성했다.

## 현재 제공되는 기능
- 학식 (미래백년관, 밀레니엄관)

## 추후 제공 기능
- 학교 공지사항


## Version
```
python version 3.4.3  

```

## How It Works
git clone

```sh
$ git clone https://github.com/ndjman7/DeerMyDear.git
```
